Shows the last xx topics of your BBpress forum on your Wordpress blog
 
== Description ==

Shows the last xx topics of your BBpress forum on your Wordpress blog. This plugin consists out of two widgets one contains the last topics created and the second one the last posts made. 

== Installation ==

1. Upload the file `bbpress-last-topics.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Change your settings under Settings -> BBpres last topics
1. In case you use the wordpress blog in is installed under the same database as your BBpress forum you can check the checkbox. Otherwise you have to enter the data information for the database.
1. BBpress Prefix is the prefix used for the BBpress tables
1. BBpress URL is the URL used for the BBpress forum.
